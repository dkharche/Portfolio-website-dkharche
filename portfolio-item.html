Stroke PredictionDeployment

flask app deployment using AWS

The "Stroke Prediction Web Application" is a cutting-edge Flask-based project designed to accurately assess an individual's probability of experiencing a stroke. Users can easily input essential health parameters, such as age, gender, marital status, glucose level, BMI, and smoking status, through a user-friendly interface. The machine learning model, deployed within the Flask app, processes this information and generates a precise prediction of whether the likelihood of a stroke is positive or negative.
To ensure data privacy and user management, the application incorporates a robust login and registration system, safeguarding sensitive user data. The collected user information is securely stored in MongoDB, providing seamless data retrieval and analysis for future use. Furthermore, the "Stroke Prediction Web Application" boasts impressive scalability and performance by deploying it on an AWS EC2 instance. The app leverages Nginx as a high-performance web server, efficiently handling incoming requests.
Additionally, Gunicorn serves as the WSGI server, efficiently interfacing with Flask to process user inputs and generate predictions. The integration of AWS's cloud infrastructure ensures optimal performance and reliability, catering to a growing number of users.
