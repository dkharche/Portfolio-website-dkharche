Stroke PredictionDeployment

flask app deployment using AWS

The Stroke Prediction with AWS Deployment Project is an end-to-end project that uses machine learning and AWS cloud services to develop a predictive model to identify individuals at higher risk of stroke. The project aims to provide a platform to analyze and visualize a patient's medical data and predict the risk of stroke. The project uses AWS services such as Amazon EC2 to build and deploy the predictive model. The model will be built using supervised machine learning algorithms such as Random Forest and will be trained and tested using a dataset consisting of patient medical data.

The model will be deployed on AWS and made available to medical professionals and patients through a web application. The web application will include interactive visualizations of the model results and will enable users to input data and receive personalized predictions. The project will be implemented using Python ,flask,HTML and CSS and AWS services
